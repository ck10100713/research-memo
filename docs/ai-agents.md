## 專案概述

這是黃佳老師《動手做AI Agent》一書的配套程式碼倉庫。黃佳是知名的 AI 技術作者，著有《GPT圖解》、《機器學習》、《數據分析》等多本暢銷技術書籍。

這個專案的核心價值是提供一個實作導向的 AI Agent 學習資源，讓讀者能夠透過實際的程式碼範例，從零開始理解和建構 AI Agent。配合書籍閱讀，可以獲得完整的理論與實作知識。

適合場景：
- 想要系統性學習 AI Agent 的中文讀者
- 需要中文技術書籍配套程式碼的學習者
- 對 LLM 和 Agent 技術有興趣的開發者
- 尋找中文 AI 學習資源的入門者

## 核心功能

1. **書籍配套程式碼**：與《動手做AI Agent》書籍章節對應的完整程式碼
2. **實作範例**：從基礎到進階的 Agent 實作案例
3. **中文說明**：全中文的程式碼註解和說明
4. **系列書籍支援**：與作者其他書籍（GPT圖解、機器學習）形成知識體系

## 研究心得

黃佳老師的《動手做AI Agent》是目前中文市場上少數系統性介紹 AI Agent 的書籍，對於中文讀者來說是非常珍貴的學習資源。

**專案特色：**
1. **中文原創**：完全針對中文讀者撰寫，避免翻譯的理解障礙
2. **實作導向**：每個概念都有對應的程式碼實作
3. **知識體系**：與作者其他書籍形成完整的 AI 學習路徑
4. **暢銷作者**：黃佳在 AI 技術書籍領域有良好口碑

**對 AI Agent 開發者的價值：**
- 系統性理解 Agent 的設計原則
- 獲得可運行的程式碼範例
- 建立完整的 AI 知識體系

**注意事項：**
- GitHub 倉庫主要是書籍配套，建議搭配書籍閱讀
- README 內容較簡潔，詳細說明在書籍中

**推薦給：**
- 偏好中文學習資源的開發者
- 喜歡實體書籍的學習者
- 需要系統性 AI 知識的入門者
