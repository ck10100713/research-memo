## 專案概述

Insta-Booster 是一個自動化工具，用於增加 Instagram Reels 的觀看次數和按讚數。它透過 Zefame API 來實現自動化的社群媒體互動增強功能。

這個專案展示了一個簡單的 Python 自動化工具如何與外部 API 互動來完成特定任務。雖然功能相對單一，但其設計模式（配置檔、選單介面、API 整合）對於理解自動化工具的開發有參考價值。

**重要警告**：此工具僅供教育目的。過度或濫用可能違反 Instagram 服務條款，請自行承擔風險。

適合場景：
- 學習 Python 自動化工具開發
- 理解 API 整合模式
- 參考 CLI 選單介面設計
- **不建議用於實際違反平台規則的行為**

## 核心功能

1. **Reels 觀看數增強**：增加 Instagram Reels 的觀看次數
2. **按讚數增強**（v0.0.2 新增）：增加 Reels 或貼文的按讚數
3. **可配置的增強數量**：自訂每次執行的增強數量
4. **配置檔管理**：使用 JSON 儲存設定
5. **互動式選單**：終端機介面的操作選單

## 安裝與使用

### 前置需求

- Python 3.12+
- pip

### 安裝方式

```bash
# 克隆專案
git clone https://github.com/Sneezedip/Insta-Booster.git
cd Insta-Booster

# 安裝依賴
pip install colorama requests
```

### 使用方式

```bash
# 執行程式
python main.py

# 選單選項：
# 1. Start Boost - 開始增強
# 2. Change Video URL - 設定 Instagram Reel 連結
# 3. Change Boost Amount - 設定增強數量
# 4. Exit - 離開程式
```

### 配置檔

```json
{
  "video_url": "https://instagram.com/reel/xxxx",
  "amount_of_boosts": 100
}
```

## 研究心得

Insta-Booster 是一個非常簡單的 Python 自動化工具，雖然功能有限，但對於學習自動化工具開發有一定的參考價值。

**技術觀察：**
1. **極簡架構**：只需要 colorama 和 requests 兩個套件
2. **配置模式**：使用 JSON 檔案持久化設定
3. **CLI 選單**：簡單的互動式終端介面

**學習價值：**
- 理解 API 請求的基本模式
- 學習配置檔的讀寫處理
- 參考 CLI 選單的設計方式

**對 AI Agent 開發的參考：**
- 展示了最簡化的自動化工具架構
- 配置管理模式可用於 Agent 設定
- **但不建議參考其功能目的**

**重要警告：**
1. 這類工具可能違反 Instagram 服務條款
2. 濫用可能導致帳號被封禁
3. 此研究純粹從技術架構角度進行，不鼓勵實際使用
4. 建議僅作為自動化工具開發的學習參考

**不建議的原因：**
- 違反平台規則
- 可能涉及虛假互動
- 長期來看對使用者無實際價值
