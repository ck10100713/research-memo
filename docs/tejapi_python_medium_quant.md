## 專案概述

TEJAPI Python Medium Quant 是台灣經濟新報 (TEJ) 在 Medium 上發布的量化分析系列文章配套程式碼。這個專案包含了從基礎的定期定額回測到進階的機器學習預測、ESG 因子選股等多種量化分析範例。

這個專案的核心價值在於提供台灣股市的量化分析實作範例，使用 TEJ API 取得專業級的金融數據。對於想要進行台股量化研究的開發者來說，這是非常珍貴的中文學習資源。

適合場景：
- 台灣股市量化分析研究
- 學習 Python 金融數據分析
- TEJ API 使用範例參考
- 技術指標回測實作

## 核心功能

1. **定期定額回測**：以 0050 為標的的定期定額策略回測
2. **技術分析回測**：MACD、KD、RSI 等指標回測實戰
3. **機器學習預測**：使用 ML 模組預測市場
4. **籌碼分析**：三大法人交易策略分析
5. **事件研究法**：現金增資宣告效果研究
6. **效率前緣**：投資組合效率前緣曲線
7. **ESG 因子選股**：ESG 量化因子策略
8. **風險值計算**：投資組合風險值 (VaR)
9. **產業動態指數**：台電產業動態指數應用
10. **盈餘管理**：衡量公司盈餘管理程度

### TEJ API 基本使用

```python
import tejapi

# 設定 API 金鑰
tejapi.ApiConfig.api_key = "YOUR_API_KEY"

# 取得證券交易資料
data = tejapi.get('TWN/EWPRCD', 
                  coid='2330',  # 台積電
                  mdate={'gte':'2020-01-01','lte':'2023-12-31'},
                  paginate=True)
```

## 研究心得

TEJAPI Python Medium Quant 是台灣量化分析領域少見的系統性學習資源，結合了 Medium 文章的理論說明和 GitHub 的實作程式碼。

**專案特色：**
1. **台股專精**：使用 TEJ 專業級台股數據
2. **系統完整**：從基礎到進階共 18+ 篇文章
3. **中文說明**：完全中文的文章和程式碼註解
4. **實戰導向**：每篇文章都有可執行的程式碼

**學習價值：**
- 學習使用 Python 進行量化分析
- 理解各種技術指標的實作
- 了解 TEJ API 的使用方式
- 掌握回測框架的基本概念

**對 AI Agent 開發的參考價值：**
- 展示如何整合專業金融數據 API
- 技術指標計算邏輯可用於 Agent 分析功能
- 回測框架設計可參考用於 Agent 決策驗證

**限制與注意：**
- 需要 TEJ API 訂閱（付費）
- 部分程式碼可能需要更新以配合 API 變更
- 適用於台股市場，非全球通用

**推薦給：**
- 想要學習台股量化分析的開發者
- 已有 TEJ 訂閱的使用者
- 需要中文量化學習資源的研究者
